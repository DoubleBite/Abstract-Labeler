Labeling Tool Usage

1. Load the data

按"選擇檔案"載入標註 txt 檔(可一次載入多個)。File list 可以挑選剛剛載入的檔案。

2. Label the data

點擊中間的綠/藍/紅色按鈕可以將滑鼠選取的字句加上該 label 的顏色，有關標註 data 與 label 的解釋請參考 tmp。
工具列右方的功能鍵依照順序分別可以清除滑鼠選取區域的顏色樣式、跳回上一步、跳回下一步。

3. Select paper type and confidence

最後一個標註程序，必須要選擇 Paper 的種類(左下角)，以及您對這篇標註的信心水準(中間下方)，這兩項標籤將有助於我們進行後續的深度學習分析。
全部的標註都完成後，即可按下右下角的 Submit Button。 Submission 程序會請求您的 Google 信箱授權，以便讓我們用信箱名稱來記錄您的標註歷史。

4. Wait for server response

上傳完畢之後，網頁會跳出通知，如下圖。如果沒有通知請耐心稍後或重新上傳一次。
另外，測試者在上傳完三篇之後應會跳出總分，並告知是否通過測試，如果沒有跳出總分(只有跳出個別分數)也請重新上傳一次。

※註：本標註程式使用google提供的firebase作為雲端空間。只要您在本機端做好所有操作並且存檔，便會自動上傳答案至指定雲端空間，不需要額外做繳交的動作。你只需要確認同時存有在本機端的同名檔案備份即可。因此建議您將所有標註過的論文放在同一資料夾內，以便日後有需要與您核對用。
